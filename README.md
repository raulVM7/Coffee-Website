# Brew Haven — Coffee Shop Website

A responsive landing page for a fictional artisan coffee shop, built from scratch to practice modern layout and responsive design.

## Built with
- HTML5
- CSS3 (Flexbox, Grid, custom properties)
- JavaScript (mobile menu)

## Features
- Fully responsive (mobile, tablet, desktop)
- Hamburger navigation on mobile
- Hero, product cards, and footer sections
